## Demo

1. Install dependencies:
  ```bash
  npm install
  ```

2. Run mongodb:
  ```bash
  npm run mongodb:production
  ```

3. Run redis:
  ```bash
  npm run redis:production
  ```

4. Go to [localhost:3000/demo](http://localhost:3000)

## Development

1. Install dependencies:
  ```bash
	npm install
  ```

2. Run mongodb:
  ```bash
  npm run mongodb:development
  ```

3. Run redis:
  ```bash
  npm run redis:development
  ```

4. Run dev:
  ```bash
  npm run dev
  ```

## Build

  ```bash
	npm run build
  ```
